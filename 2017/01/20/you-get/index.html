You-Get是一个简单的可以从web上获取多媒体文件（音频，视频，图片）的命令行程序，方便小巧。

默认下载路径是C:\user\yourPCname\下的。如上所示，-o D:\IDM去掉就下载到默认文件夹了。

使用说明

window环境下需要安装python（建议最新稳定版）

安装说明

  1. git clone $ git clone git://github.com/soimort/you-get.git

  2. 使用python包管理器 pip pip3 install you-get

升级You-Get

$ pip3 install –upgrade you-get

其他

  1. 下载视频时可以选择画质以及格式。
  2. URL可以直接使用。也可以直接输入网页地址自动检测内容。

快捷键

CTRL+C 暂停下载，下次再次下载已暂停的文件时，将从上次暂停的地方开始。

支持网站

国内很多网站都支持。
