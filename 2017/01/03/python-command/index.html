pip常用命令

xiaoge Jan 03, 2017
  • 记录

查看已安装的包

pip freeze / pip list

安装包

在线安装

pip install SomePackage/ pip install -r quirements.txt

安装包示例
pip install you-get

安装某个版本的you-get
pip install you-get==0.4.026

requirements.txt里的内容格式为:
PySocks==1.6.5
requests==2.12.4
six==1.10.0
xmltodict==0.10.2
you-get==0.4.626

本地安装包

pip install 路径+SomePackage

卸载包

pip uninstall SomePackage

升级包

pip install Somepackage==版本号 # specific version

升级pip自己

pip install upgrade pip

查询可升级的包

pip list –outdate

FAQ

某次更新后查看安装包时出现这样的提示:

DEPRECATION: The default format will switch to columns in the future. You can use –format=(legacy|columns) (or define a format=(legacy|columns) in your pip.conf under the [list] section) to disable this warning.

就是说以后的pip list的默认格式就会采用columns.
解决方案:
创建 %APPDATA%\pip\pip.ini 文件，添加如下文件内容：
[list]
format=columns
