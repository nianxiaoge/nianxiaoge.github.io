windows 下命令行工具 Cmder

如果不做开发的话，在 windows 下用命令行的地方很少，偶尔也就是 ping 个网络而已。复杂的功能自带的 cmd 并不能实现，包括 powershell 。

而且它们都不好看（感觉这个才是重点）。

这个 cmder 也是在网络上发现的，功能很强大的命令行工具。Windows环境下非常简洁美观易用的cmd替代者，

cmder 是官网。下载选项有两个，一个是 Download mini, 一个是 Download full。

mini portable 版，沒有内置 msysgit 工具。

full 也是 portable 版，不过功能就强大多了。自带了 msysgit, 除了这个命令之外, 它支持了很多Linux命令。

cmder 安装使用

因为我的笔记本上已经装了 git, 所以下载了 mini 版，这里也仅介绍 mini 版。

解压缩到某个文件夹（我选择的是C:\Program Files (x86)），然后在系统变量里的 Path ，把 cmder 所在的路径添加进去就好了。

然后 win + r 输入 cmder ,就可以打开了。

cmder 配置

windows 下的 cmd 可以在任意目录下按住 shift 键，同时鼠标右键呼出 cmd 窗口。
如果我们也想在任意目录下打开 cmder 的话，可以
先以管理员权限打开 cmd ,输入 Cmder.exe /REGISTER All 回车。此时 cmder 已经加到系统右键菜单了。

以管理员身份运行

打开 cmder, Ctrl + t 的快捷键，勾选上下面的
Run as administrator。

设置

可以 win + alt + p 快捷键呼出 cmder 的设置界面，也可以在顶部或者底部鼠标右键就看到 settings 了。

设置里面有相当多的项，可定制化非常高。
