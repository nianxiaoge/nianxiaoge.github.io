Vim编辑器初步上手

xiaoge Feb 20, 2017
  • course

前言:

Vim是从vi发展出来的一个终端文本编辑器。代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。和Emacs并列成为类Unix系统用户最喜欢的编辑器。维基百科

vim

vim 有三种模式

  1. 在光标当前位置输入 i 即进入编辑模式
    左下角出现 -insert- 此时就可以跟其它编辑器一样开始写代码了
    此时键盘输入直接被写进了文件里，当前vim命令都失效了。

  2. 在光标当前位置输入 v 即进入 visual模式
    左下角出现 -visual- 一般在需要选择文本时进入该模式

  3. 在光标当前位置输入 ctrl+v 即进入 visual block模式
    该模式常用于快速注释代码

  4. 退出当前模式皆为 esc键

git bash vim

  1. git bash输入vim即可进入vim模式
  2. 如果是输出状态，首先Esc退出输入状态，然后Shift+;，左下角会有 : 出现
    再输入q!或wq!（不保存改动，wq!是保存文件的写入修改）退出。

Q:
windows下 git bash 打开vim 显示乱码解决：
任一目录打开git bash 输入：cd /etc 进入etc目录

输入 vi vimrc
在首行的下面输入：

set nu

set fencs=utf-8,gbk,utf-16,utf-32,ucs-bom

保存后退出再次打开git bash进入vimrc文件就好了。
