Markdown初入门

xiaoge Nov 07, 2016

Markdown

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。(摘自维基百科)
用户可以使用标记符号以最小的输入代价生成极富表现力的文档。它使用简单的符号标记不同的标题，分割不同的段落 粗体 或者 斜体 某些文字。

基本语法

标题
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

定义标题的话在标题前面加上 # 标题名。
如果你知道html的话，就会明白，上面六个标题等同于 h1-h6 的效果。

显示为

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

在 # 和标题之间有一个空格,这是它的语法规定的。

列表
- 列表一
- 列表二
- 列表三

显示为：

  • 列表一
  • 列表二
  • 列表三

这是无序列表。
加上1，2，3即为有序列表。

1 列表一
2 列表二
3 列表三

显示为：

1 列表一
2 列表二
3 列表三

引用
> 我是被引用的内容,在我前面加上大于号就行了

显示为：

我是被引用的内容,在我前面加上大于号就行了

插入链接，图片

[谷歌](https://google.com.hk)

显示为

谷歌

图片
![图片名称](图片URL)

显示为：

粗体与斜体

粗体

**文字开始与结束的左右有两个***

显示为：

文字开始与结束的左右有两个*

斜体：

*斜体就是一个**

显示为：

斜体就是一个*

代码段

hello world //我的左边有四个空格，即Tab键

显示为：

hello world //我的左边有四个空格，即Tab键
code写法
<code>hello world</code>

显示为：

hello world
